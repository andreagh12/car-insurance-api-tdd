# Car insurance quote calculator API

Backend API for car insurance quote calculation, built using Node.js and Express with Test-Driven Development (TDD).

## Tech used
- Node.js  
- Express  
- CORS  
- Jest  
- Supertest  

## Overview
This project was developed as part of a training programme in a team environment.  

The overall system included multiple APIs for an insurance purchasing process. In this repository, I focused on building the insurance quote API, which calculates premiums based on car value and risk rating.

## Features
- REST API endpoint for insurance quote calculation  
- Monthly and yearly premium calculations  
- Input validation and error handling  
- Automated unit and integration testing
